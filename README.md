# 🌟 Pokedex App 🌟

¡Bienvenido a la Pokedex App! Tu guía definitiva para explorar el fascinante mundo Pokémon. Descubre información detallada sobre tus Pokémon favoritos, sus habilidades, tipos, evoluciones y mucho más.

Este proyecto ha sido desarrollado con ❤️ por **SebCode Academy**.

## ✨ Características Principales

*   📱 **Interfaz Moderna y Amigable**: Navega fácilmente por la información de los Pokémon.
*   🔍 **Búsqueda Avanzada**: Encuentra Pokémon por nombre, número o tipo.
*   📊 **Datos Completos**: Accede a estadísticas, habilidades, evoluciones y descripciones de la Pokedex.
*   🎨 **Diseño Atractivo**: Una experiencia visual agradable para todos los entrenadores Pokémon.
*   🔄 **Actualizaciones Constantes**: (Opcional, si planeas actualizarla) Nuevos Pokémon y características añadidas regularmente.

## 🚀 Instalación

Sigue estos sencillos pasos para tener la Pokedex App funcionando en tu entorno local.

### Requisitos Previos

*   **Node.js**: Asegúrate de tener Node.js instalado. Puedes descargarlo desde [nodejs.org](https://nodejs.org/). Se recomienda la versión LTS más reciente.
*   **npm** o **yarn**: Gestor de paquetes (npm viene con Node.js). Yarn es una alternativa popular.

### Pasos para la Instalación

1.  **Clona el Repositorio:**
  Abre tu terminal o línea de comandos y ejecuta:
  ```bash
  git clone https://github.com/Sebas1124/pokedex_animated_useFetch
  ```

2.  **Navega al Directorio del Proyecto:**
  ```bash
  cd pokedex_animated_useFetch
  ```

3.  **Instala las Dependencias:**
  Usa npm:
  ```bash
  npm install
  ```
  O si prefieres yarn:
  ```bash
  yarn install
  ```

4.  **Inicia la Aplicación:**
  Para ejecutar la aplicación en modo de desarrollo:
  Usa npm:
  ```bash
  npm start
  ```
  O si prefieres yarn:
  ```bash
  yarn start
  ```
  Esto generalmente abrirá la aplicación en tu navegador web predeterminado en una dirección como `http://localhost:3000` (el puerto puede variar).

## 🛠️ Uso

Una vez que la aplicación esté en funcionamiento:

1.  Explora la lista de Pokémon.
2.  Usa la barra de búsqueda para encontrar Pokémon específicos.
3.  Haz clic en un Pokémon para ver sus detalles.
